# RabbitEmergency.com Priority 50 improvement manifest

Source file for `tools/publish_priority50_improvements.py`.

Rules for this manifest:
- One row per Priority 50 owner or group.
- Keep language vet-first, source-cited, and claim-safe.
- `go_now`, `call_today`, `do_not`, `tell_vet`, `nuance`, `vet_checks`, and `faq` use semicolon-separated items.
- `siblings` use comma-separated slugs that already exist as live pages.
- Publish in batches of 20 with the generator, then run the consolidation guard.
